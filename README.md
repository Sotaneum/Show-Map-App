# Show Map

위치정보로 지도위에 마커를 표시합니다.

![cover](./assets/video.gif)

## Info

- 주소 검색을 통해서 지도 위에 표시합니다.
- 위경도 값으로도 지도 위에 표시합니다.
  
## Finally

- 안드로이드 공부를 하면서 구현한 앱입니다.
